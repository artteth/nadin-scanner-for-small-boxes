Как подключить:
1. Нажмите кнопку ᯤ внизу справа
2. Нажмите «Подключить сканер»
3. Выберите свой сканер в диалоге браузера
4. Сканируйте штрих-коды — они появятся здесь

Браузеры: Android Chrome/Edge, iOS → Bluefy

🔧 Сканер привязан к iOS?
  1. Настройки → Bluetooth → ⓘ рядом со сканером
  2. «Забыть устройство»
  3. Вернитесь, нажмите «Подключить»
  4. На вопрос «Создать пару» → Отменить
iOS: браузер Bluefy  |  Android: Chrome / Edge
