# Функция сохранения дат упаковки (v148+)

## Описание

Вместо сохранения простой галочки (TRUE/FALSE) в Google Sheets, система теперь записывает дату и время, когда пользователь отметил коробку как упакованную. Это позволяет отслеживать, когда именно произошла упаковка.

## Формат данных

- **Дата упаковки**: `YYYY-MM-DD` (например: `2025-02-15`)
- **Столбец в Google Sheets**: AQ (колонка 43) в листе "WB"
- **Старый формат** (совместимость): TRUE/FALSE или 1/0 всё ещё поддерживаются

## Как это работает

### На веб-странице

1. **Страница "Сканер упаковки"** (`index.html`):
   - Пользователь открывает модальное окно "Все изделия"
   - В таблице видит галочку для каждого изделия
   - Если изделие уже упаковано, вместо пустой галочки показывается **дата + галочка**
   - При клике на галочку система сохраняет текущую дату в формате `YYYY-MM-DD`

2. **Страница "Отметка упаковки"** (`отметка упаковки страничка.html`):
   - Список всех изделий со статусом упаковки
   - Для новых записей (с датой) показывается дата рядом с галочкой
   - Для старых записей (TRUE/FALSE) показывается просто галочка
   - При снятии галочки дата очищается

### На сервере (Google Apps Script)

**Файл: `server-code.js`**

```javascript
// Форматирование даты
function formatDate(date) {
  const pad = (n) => n < 10 ? '0' + n : n;
  const year = date.getFullYear();
  const month = pad(date.getMonth() + 1);
  const day = pad(date.getDate());
  return year + '-' + month + '-' + day;
}

// При сохранении (функция saveItem)
if (repacked === true) {
  const now = new Date();
  const timestamp = formatDate(now);
  sheet.getRange(rowNum, 43).setValue(timestamp); // AQ
} else {
  sheet.getRange(rowNum, 43).setValue(''); // очистить
}
```

## Поддержка старых данных (Backward Compatibility)

Система автоматически распознаёт оба формата:

- **Новый формат**: если значение — строка, соответствующая regex `/^\d{4}-\d{2}-\d{2}/`
  - Пример: `"2025-02-15"`
  - Отображается как: `2025-02-15` + ☑️

- **Старый формат**: если значение — `TRUE`, `true`, `1` или boolean `true`
  - Отображается как: ☑️ (только галочка)

Функция `isPacked()` проверяет оба варианта:

```javascript
function isPacked(v) {
  // Новый формат: непустая дата-строка
  if (typeof v === 'string' && v.trim().length > 0) return true;
  // Старый формат: boolean или числовые значения
  return v === true || v === 1 || v === 'TRUE' || v === 'true';
}
```

## Функции в коде

### JavaScript (Фронтенд)

#### `isDateFormat(v)`
Проверяет, является ли значение датой в формате YYYY-MM-DD.
```javascript
function isDateFormat(v) {
  if (!v || typeof v !== 'string') return false;
  const s = v.trim();
  return /^\d{4}-\d{2}-\d{2}/.test(s);
}
```

#### `formatTimestamp(ts)`
Форматирует дату для отображения (убирает пробелы).
```javascript
function formatTimestamp(ts) {
  if (!ts || typeof ts !== 'string') return '';
  return ts.trim();
}
```

#### `isPacked(v)`
Проверяет, упакована ли коробка (поддерживает оба формата).

### Google Apps Script (Бэкенд)

#### `formatDate(date)`
Преобразует JavaScript Date в строку `YYYY-MM-DD`.

#### `saveItem(model, repacked, packType)`
Сохраняет статус упаковки с датой (если `repacked === true`) или очищает поле (если `false`).

#### `saveData(updates)`
Массовое сохранение. Для каждого изделия со статусом `repacked: true` сохраняет текущую дату.

## Миграция данных

При переходе на эту версию:
- Старые записи (TRUE/FALSE) продолжают работать
- Новые отметки записываются как даты
- Постепенно база данных переходит на новый формат
- Нет необходимости в миграции, всё работает автоматически

## Пример в Google Sheets

| Модель | ... | AQ (Упаковано) | AS (Коробка) |
|--------|-----|--------|------|
| 1-11711/1 | ... | 2025-02-15 | 5см коробка M |
| 2-22222/0 | ... | TRUE | 3см коробка S |
| 3-33333/1 | ... | (пусто) | — |

- **Строка 1**: новый формат — дата упаковки
- **Строка 2**: старый формат — совместимость
- **Строка 3**: не упакована

## Тестирование

1. Откройте "Все изделия"
2. Отметьте упакованную коробку галочкой
3. Проверьте Google Sheets — должна появиться дата `YYYY-MM-DD`
4. Снимите галочку — дата должна удалиться
5. Старые записи (TRUE) должны отображаться как галочка
